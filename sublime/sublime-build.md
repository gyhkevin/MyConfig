# sublime-bulid on Mac

## PHP Build
```json
{
   "cmd": ["php", "$file"],
   "selector": "source.php",
   "file_regex": "php$"
}
```

## JavaScript Build 
```json
{
   "cmd": ["node", "$file"],
   "selector": "source.js",
   "file_regex": "js$"
}
```
